Sobre Mim

Sou estudante de Análise e Desenvolvimento de Sistemas, com foco em evolução técnica contínua e aprendizado sólido de stacks modernas.

Tenho interesse em arquitetura de software, boas práticas, testes automatizados e organização modular. Gosto de trabalhar na lógica por trás dos sistemas, entendendo comportamento, fluxo de dados e regras de negócio além da interface.

Busco desenvolver sistemas seguros, de fácil manutenção e baseados em princípios de arquitetura limpa. Estou em constante aprimoramento técnico, com foco em qualidade de código, clareza estrutural e evolução sustentável do software.

Portfólio

Infra Monitoring System

Projeto didático focado em pipelines e monitoramento: coleta métricas do host via psutil, publica métricas para scraping pelo Prometheus, empacota serviços com Docker para ambientes reproduzíveis e usa GitHub Actions para executar testes, análises de segurança e checks de qualidade.

JobManage

Sistema de orquestração de jobs com controle de concorrência por lease, máquina de estados, retry com backoff jittered e coordenação via banco de dados. Construído com FastAPI, SQLite, Alembic, testes automatizados e CI no GitHub Actions, com métricas, health checks e logs estruturados
